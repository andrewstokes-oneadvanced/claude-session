TYPICAL WORKFLOW
================

1. FIRST TIME SETUP:
   - Run setup.bat
   - Application builds and offers to start
   - Icon appears in system tray

2. DAILY USE:
   - App runs automatically (if startup enabled)
   - Hover icon to see session time and tokens
   - Right-click for menu options
   - Commands sent every 5 hours automatically

3. CUSTOMIZATION:
   - Edit TrayApplicationContext.cs
   - Change interval, command, or features
   - Run build.bat to rebuild
   - Restart application

4. TROUBLESHOOTING:
   - Check USER_GUIDE.md
   - Verify Claude Code CLI works: echo hi | claude
   - Right-click icon → Send 'hi' Now to test
